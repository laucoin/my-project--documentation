# C4 Model

The architecture is documented using the C4 model, which provides successive levels of zoom into the system.

| Level              | Page                                                 | Audience                                                          |
|--------------------|------------------------------------------------------|-------------------------------------------------------------------|
| 1 – System Context | [Context](/technical/architecture/c4/context)        | Anyone — what the system does and who uses it                     |
| 2 – Containers     | [Containers](/technical/architecture/c4/containers)  | Architects, leads — the deployable units and how they communicate |
| 3 – Components     | [Components](/technical/architecture/c4/components/) | Developers — the internal structure of each container             |

Diagrams are written in [Mermaid C4 syntax](https://mermaid.js.org/syntax/c4.html).
