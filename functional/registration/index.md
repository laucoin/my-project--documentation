# Registration

The Registration section covers how people can request to join a project and how those requests are managed.

::: info Option required
Registration is only available if the **REGISTRATION** option is enabled on the project.
:::

## Contents

- [Registration Period](/functional/registration/registration-period) — how a project opens itself to registration
  requests
- [Registration Form](/functional/registration/registration-form) — the custom form attached to a period and filled in
  by requesters
- [Registration Request](/functional/registration/registration-request) — how a user submits a request and how it is
  processed
