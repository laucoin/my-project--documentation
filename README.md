## This repository 📖

This project contains documentation for a personal project made available free of charge to an organization (Scouts et
Guides de France).

It uses [VitePress](https://vitepress.dev/) and [pnpm](https://pnpm.io/).

## How to install and use it? ⚙️

### Prerequisites

Install [Node.js 22 or later](https://nodejs.org/en/download)

Install [pnpm](https://pnpm.io/installation)

### Build and run locally

##### Procedure

1. Clone this repository with:
    ```shell
    git clone https://github.com/laucoin/my-project--documentation.git
    ```
   OR
    ```shell
    git clone git@gitlab.com:laucoin/my-project--documentation.git
    ```
2. Move into the documentation directory
    ```shell
    cd my-project--documentation/
    ```
3. Install dependencies
    ```shell
    pnpm install
    ```
4. Enjoy the following commands 🎉

#### Running the application in dev mode

You can run the documentation in dev mode that enables live reload using:

```shell
pnpm dev
```

#### Build the documentation

The documentation can be built using:

```shell
pnpm build
```

It produces the static files in the `.vitepress/dist/` directory.

#### Preview the built documentation

You can preview the built documentation using:

```shell
pnpm preview
```
